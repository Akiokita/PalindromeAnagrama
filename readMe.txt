Para verificar se existe algum anagrama de um palíndrome dentro de uma cadeia de caracteres,
devemos realizar os seguintes passos:
- Criar uma lista com todas as permutações da cadeia de caracteres desejada.
- Percorrer todas as permutações da lista e verificar se cada uma delas é um palíndrome.
- Caso exista pelo menos uma opção que seja um palíndrome, retorna verdadeira.
- Caso contrário retorna falso.
